Plimsolls Key Features

  1. Built on Plim, Shower and beautiful Python
  2. Build-able on local and deploy-able on remote (e.g. GitHub Page)
  3. Programmable presentation
  4. Tools supported:
    Font Awesome, SCSS, Markdown, Restructured Text, and Pygments

Inherited key features of Shower and Plim
